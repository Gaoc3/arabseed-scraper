# 🚀 دليل رفع المكتبة إلى PyPI (PyPI Publishing Guide)

هذا الدليل يشرح لك خطوة بخطوة وببساطة كيف ترفع هذه المكتبة الرائعة لتصبح متوفرة للجميع على مستودع حزم بايثون الرسمي (PyPI) بحيث يمكن لأي شخص تحميلها بكتابة `pip install arabseed-scraper`.

---

## 📋 المتطلبات القبلية (Prerequisites)

1. **حساب على PyPI**: 
   إذا لم يكن لديك حساب، قم بالتسجيل مجاناً في دقيقة واحدة هنا: [pypi.org/account/register](https://pypi.org/account/register/)
   
2. **إنشاء رمز وصول آمن (API Token)**:
   - بعد تسجيل الدخول، اذهب إلى إعدادات الحساب (Account Settings).
   - انزل لأسفل الصفحة واضغط على **Add API token**.
   - اختر اسماً للرمز (مثال: `arabseed-scraper-token`) وحدد الصلاحيات لتكون شاملة للحساب بأكمله (Unlimited scope) إذا كان هذا أول مشروع ترفعه.
   - انسخ الرمز الذي يبدأ بـ `pypi-` واحتفظ به في مكان آمن (لن يظهر لك مرة أخرى).

---

## 🛠️ خطوات الرفع (Step-by-Step Publishing)

افتح سطر الأوامر (Terminal / CMD) في مجلد المشروع:
```bash
cd "C:\Users\secon\Documents\ArabSeedScraper"
```

### الخطوة 1: تثبيت أدوات البناء والرفع الرسمية
قم بتثبيت أدوات بناء وتوزيع حزم بايثون الرسمية (`build` و `twine`):
```bash
pip install --upgrade build twine
```

### الخطوة 2: بناء الحزمة (Build the Package)
قم بتشغيل الأمر التالي لإنشاء ملفات الحزمة المضغوطة (Source Distribution & Wheel):
```bash
python -m build
```
*بعد انتهاء الأمر، ستجد أنه تم إنشاء مجلد جديد باسم `dist/` يحتوي على الملفات البرمجية المهيأة للنشر.*

### الخطوة 3: التحقق من سلامة الحزمة قبل الرفع
قم بتشغيل أداة التحقق للتأكد من أن ملف التوثيق والبيانات الوصفية للحزمة متوافقة تماماً مع معايير PyPI:
```bash
python -m twine check dist/*
```
*(يجب أن تظهر لك رسالة `Passed` باللون الأخضر).*

### Step 4: الرفع الفعلي إلى PyPI (Upload)
قم بتشغيل هذا الأمر لرفع الحزمة إلى المستودع الرسمي:
```bash
python -m twine upload dist/*
```

**🔑 عند طلب البيانات الوصفية أثناء الرفع:**
*   **Username**: اكتب حرفياً: `__token__`
*   **Password**: قم بلصق رمز الوصول (API Token) الذي قمت بنسخه سابقاً (والذي يبدأ بـ `pypi-`).
    *(ملاحظة: لن ترى الرمز أثناء اللصق في الطرفية لأسباب أمنية، فقط قم بلصقه واضغط Enter).*

## 🎉 مبروك!
المكتبة الآن مرفوعة بنجاح! يمكن لأي شخص في العالم تثبيتها واستخدامها في كود البايثون الخاص به بكل بساطة بكتابة:
```bash
pip install arabseed-scraper
```

وتشغيلها تفاعلياً من أي سطر أوامر بكتابة:
```bash
arabseed-scraper
```
